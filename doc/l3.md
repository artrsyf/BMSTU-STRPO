# Отчёт о выполнении ЛР№3

## Наименование работы

Формирование спецификации OpenAPI.

## Спецификация OpenAPI

Спецификация OpenAPI сформирована в папке `oas` проекта.

## Выводы

Работа была выполнена для изучения принципов проектирования REST API и описания микросервисов с использованием спецификации OpenAPI (Swagger) на основе выделенных bounded contexts и пользовательских историй. В процессе выполнения работы были приобретены навыки проектирования API, описания HTTP-методов, маршрутов, схем данных и интеграционного взаимодействия между сервисами в архитектуре, построенной с использованием подхода Domain-Driven Design.
